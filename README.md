# 🚀 Projects

### OpenTripPlanner

OpenTripPlanner는 “단순 장소 추천”이 아닌 일정을 세워 주는 스마트 여행 지원 서비스입니다. 지역·날짜·이동수단만 입력하면 맞춤 장소 추천, 동선 최적화, 혼잡도 예측, 여행 기록 자동화가 한 번에 이루어지고, 사용자에게는 지도 위 동선과 타임라인 일정표로 전달됩니다. 웹·iOS·Android를 하나의 코드베이스(React Native + Expo)로 지원하는 프로젝트.

기술

ReactNative · TS · Expo · FastAPI · Python · Postgresql
